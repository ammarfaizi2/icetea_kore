# Ice Tea Kore
Web application platform and scalable web APIs in C.

# Config files
- Config `app/conf/app.conf`
- Build Config `app/conf/build.conf`

# Installation
```shell
make
sudo make install
cd app
cp app/conf/app.conf.example app/conf/app.conf
kodev run
```
https://localhost:8000 (must be https)
